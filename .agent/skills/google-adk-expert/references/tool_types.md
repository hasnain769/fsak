# ADK Tool Categories

## 1. Function Tools
Wrap Python functions with `FunctionTool`. Ensure type hints and docstrings are present for LLM documentation.

## 2. Agent Tools
Wrap another agent as a tool with `AgentTool`. Enables hierarchical delegation patterns.

## 3. MCP (Model Context Protocol)
Connect to external MCP servers using `MCPToolset`. Supports language-agnostic tool integration.

## 4. OpenAPI Tools
Auto-generate tools from REST API specs using `OpenAPIToolset`.

## 5. Enterprise Tools
- **Google Cloud**: BigQuery, Firestore, AlloyDB, etc.
- **Connectors**: 100+ enterprise integrations (GitHub, Stripe, Notion).
